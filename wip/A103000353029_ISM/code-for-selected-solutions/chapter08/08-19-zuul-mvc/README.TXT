Project: zuul-mvc
Authors: Michael Kolling and David J. Barnes
Modified by: Poul Henriksen (2005)

This project is part of supplementary material for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Seventh edition
   David J. Barnes and Michael Kolling
   Pearson Education, 2025
   
This project is a solution to Exercise 8.19.
It is a redesigned version of the original game structure, that 
illustrates how the model model-view-controller pattern can be 
applied to Zuul.
